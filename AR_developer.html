NASA SUITS Creative Design Challenge

This AR HoloLens project aimed to develop an all-in-one interface for NASA astronauts. The innovative AR UI interface was designed to be non-intrusive, displaying critical information without obstructing the view. The system included a voice control feature implemented via a language model (LLM) for hands-free operation. Additionally, we integrated a rover tracking feature for controlling Leo rovers. This project showcases my experience with AR interface design, real-time rendering, and leveraging AI for user interactions.

Key Technologies: AR, HoloLens, Voice Control, Real-Time Rendering

Role: AR Developer, Interface Designer

ShiftPixy Animation + Game (Clukin Lucky)

"Clukin Lucky" is an endless runner game developed during my time at ShiftPixy Labs. It was designed as a blend of engaging gameplay and brand promotion, where players navigated challenges and collected items. Users can choose to toggle on and off to see either the game background or the real-world background. XR Ophthalmoscopy Simulator

The XR Ophthalmoscopy Simulator is a research collaboration with the IDSC and Bascom Palmer Eye Institute, aimed at creating an immersive learning tool for medical students. This VR tool allowed students to practice eye examinations in a risk-free environment with realistic 3D eye models. I worked on UI/UX design, wireframing, and 3D modeling to develop an intuitive, user-friendly interface.

Key Technologies: VR, 3D Modeling, UI/UX Design

Role: AR/VR Developer, UI/UX Designer

Mr. Fossil

"Mr. Fossil" is an MR-based educational experience designed to teach children about paleontology in an engaging and interactive way. Utilizing MR technology, users can place and explore 3D fossil models in their real-world environment, learn fun facts, and even interact with animated dinosaurs. The app encourages a hands-on approach to learning, making the experience both educational and entertaining. This project showcases my skills in creating interactive MR content with a focus on education and user engagement.

Key Technologies: Unity, 3D Modeling, Animation
